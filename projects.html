Distributed Genetic Algorithm on Hadoop YARN

A framework which can be used to create a Genetic Algorithm program. This framework is directly written on YARN. A user needs to just write code of Genetic Algorithm operators, rest all work like - Application submission to YARN resource manager, controlling work distribution by Application master on various nodes, and there interactions are all done by framework.

Code is available on github.

Gossip protocol implementation

Gossip protocol is communication protocol used in large scale distributed systems. They are very popular because of their simplicity, scalibility, and reliability even in constantly changing environment. Core of protocol is that one node will communicate with only few nodes and inform them about what it knows. Other nodes will update there information, and share there same way to other few nodes periodically. This way after sometime all nodes will be aware of complete network. It is analgous to office gossips.

Code is available on github.
